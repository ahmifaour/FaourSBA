Welcome to the Submariner's Alexandria!


This website is dedicated to the databasing of Cold-War submarines in both the arsenals of the United States' Navy, and the Soviet Union's/Russia's Navy. This database includes: both electric-diesel and nuclear powered submarines from both nations, common armaments for each submarine class, each submarine classes' combat specifications, and prolific stories regarding each submarine class (where applied). If you are new here, please make an account! You can set your favorite submarine for all the other users to see.
